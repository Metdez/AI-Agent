# Brainstorm Custom Software Offers

Use this skill whenever the USER has a new idea for custom software (like a CRM, operations automation, or specialized AI tool) and wants to figure out how to pitch it to specific companies as a Grand Slam Offer.

## Context to Consider
1. **The Deterministic 3-Layer Architecture** (from `CLAUDE.md`): You build software by separating probabilistic routing (Layer 1/2) from deterministic execution (Layer 3 Python scripts). This guarantees high reliability—meaning you can confidently sell "zero-hallucinations in execution" while moving much faster than traditional software agencies.
2. **The 100M Offers Framework**: Value = (Dream Outcome * Perceived Likelihood of Achievement) / (Time Delay * Effort & Sacrifice)

## Execution Steps

When generating a new offer for a software idea, follow these steps sequentially:

### Step 1: Identify the Target & Outcome
- **Who is the specific company or niche?** (e.g., Real estate agencies, VC firms, e-commerce stores)
- **What is their operational bottleneck?** (e.g., Manual data entry, slow lead response times)
- **What is their dream outcome?** (e.g., Fully automated lead qualification and follow-up)

### Step 2: Map the Value Equation
Apply the 100M Offers value equation to the specific software idea:
- **Dream Outcome**: How does this customized software permanently solve their problem?
- **Likelihood of Achievement**: Emphasize the deterministic 3-layer architecture. Explain that unlike standard AI wrappers that hallucinate, we build reliable, execution-focused scripts.
- **Time Delay**: Frame the MVP delivery speed (usually 14-30 days) versus traditional agencies (6-12 months).
- **Effort & Sacrifice**: Highlight that the client only needs to provide their existing SOPs (Directives). They don't need to learn to code or manage developers.

### Step 3: Draft the Pitch
Create a 1-2 sentence core pitch. Use this format:
> *"We help [Target Audience] achieve [Dream Outcome] in [Timeframe (e.g., 14 Days)] without [Pain Point (e.g., writing code or hiring devs)] by building custom 3-layer deterministic AI software."*

**Example:** *"We turn your company's SOPs into fully autonomous, custom AI software in 14 days. You provide the SOP, we build the deterministic AI that runs it 24/7 without complaining—or you don't pay."*

### Step 4: Stack the Offer
Make the offer irresistible by adding logical enhancers:
- **Guarantee (Risk Reversal)**: e.g., "100% deterministic execution guarantee: if it doesn't execute the core directive accurately, we rewrite the execution layer for free until it does, or refund your investment."
- **Bonuses**: e.g., "Free Directive Engineering consultation to convert your SOPs into AI prompts," or "6 months of automatic self-annealing loop maintenance."
- **Scarcity/Urgency**: e.g., "Only taking 3 new implementation clients this month because we custom-code Layer 3 for every client."

### Step 5: Output Delivery
Present the finalized offer structure to the user in a clean, easily readable Markdown format, separating each element of the Value Equation, the drafted pitch, and the stacked enhancements.
